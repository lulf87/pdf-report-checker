# 前端大屏风格重设计方案

## 概述

将现有的报告核对工具前端从"现代医疗 UI"风格彻底改造为"科技感数据大屏"风格。

## 设计目标

- 深色主题 + 霓虹发光效果
- 数据仪表盘/大屏风格展示
- 粒子/网格背景动效
- 全部推倒重来，打造全新体验

---

## 1. 整体视觉风格

### 1.1 色彩系统

```
主背景:     #0a0e17 (深空蓝黑)
次背景:     #111827 (卡片背景)
边框:       rgba(59, 130, 246, 0.2) (半透明蓝)

主色调:     #3b82f6 (科技蓝)
辅助色:     #06b6d4 (青色)
强调色:     #8b5cf6 (紫色)
成功色:     #10b981 (绿)
警告色:     #f59e0b (橙)
错误色:     #ef4444 (红)

霓虹发光:
  - 主色发光: 0 0 20px rgba(59, 130, 246, 0.5)
  - 强调发光: 0 0 30px rgba(139, 92, 246, 0.6)
  - 成功发光: 0 0 15px rgba(16, 185, 129, 0.5)
```

### 1.2 字体

```css
--font-display: 'Orbitron', 'Rajdhani', sans-serif;  /* 标题/数字 */
--font-body: 'Inter', 'Source Sans Pro', sans-serif;  /* 正文 */
--font-mono: 'JetBrains Mono', 'Fira Code', monospace; /* 代码/数据 */
```

### 1.3 核心视觉效果

- **玻璃拟态 (Glassmorphism)**: `backdrop-filter: blur(12px)` + 半透明背景
- **霓虹边框**: `box-shadow: 0 0 20px rgba(59, 130, 246, 0.3)`
- **渐变发光**: 线性渐变 + 多层 shadow
- **扫描线效果**: 细微的水平线纹理增加科技感

## 3. 动效设计

### 3.1 背景动效

1. **粒子系统**: 使用 react-particles 或 tsparticles
   - 小点随机漂浮
   - 鼠标悬停时粒子聚集
   - 颜色: 蓝色/青色半透明

2. **网格背景**: CSS 渐变网格
   - 细微的网格线
   - 呼吸灯效果 (透明度变化)

### 3.2 交互动效

1. **数字跳动**: 统计数字从 0 动画到目标值
2. **卡片入场**: 依次从下方淡入
3. **状态切换**: 颜色渐变过渡
4. **Hover 效果**: 发光增强 + 微微上浮

### 3.3 数据可视化动效

1. **环形进度条**: SVG 动画从 0% 到目标值
2. **柱状图**: 从底部生长动画
3. **状态指示灯**: 呼吸灯效果
