# 打包成Mac应用说明

## 方式1：使用electron-builder打包（推荐）

```bash
# 1. 先构建React前端
npm run build:react

# 2. 打包Python后端为可执行文件
cd python_backend
pip install pyinstaller
pyinstaller --onefile --name python_backend main.py
cd ..

# 3. 打包Electron应用
npm run build:electron
```

打包后的应用在 `dist/` 目录中，可以拖动到 Applications 文件夹。

## 方式2：使用platypus创建简单包装器

安装platypus:
```bash
brew install --cask platypus
```

然后使用platypus将启动脚本包装成.app。

## 方式3：开发模式快捷启动

日常使用建议使用方法2（命令行alias）或方法1（双击.command文件）。
